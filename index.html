MY KNOWLEDGE

Day by day the tools I use to work are the following:

JIRA

Jira is an online tool for project task management, bug and issue tracking, and operational project management.

TRELLO

Trello is a project management software with a web interface and a client for iOS and Android to organize projects

GITHUB

GitHub is a forge for hosting projects using the Git version control system. It is mainly used for creating source code for computer programs.

VSCODE

Includes support for debugging, integrated Git control, syntax highlighting, smart code completion, snippets, and code refactoring

CYPRESS

Cypress is a Javascript end-to-end testing tool.In other words, it allows checking that the performance of a newly developed software product is good and corresponds to the initial needs.

SELENIUM

Selenium is a software testing environment for web-based applications.Selenium provides a record/playback tool to create tests without using a test scripting language.
